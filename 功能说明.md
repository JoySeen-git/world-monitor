## 🔧 技术架构

### 前端技术栈
- React 18 + TypeScript
- Ant Design UI
- Recharts 可视化
- Leaflet 地图
- i18n 国际化

### 后端技术栈
- Node.js + Express
- TypeScript
- WebSocket 实时通信
- lowdb JSON 数据库
- node-cron 定时任务

### 数据服务
- 免费新闻 API
- GDELT（限流）
- 世界银行数据
- 联合国数据
- 透明国际数据
- 多维度数据服务

## 📈 API 接口

### 基础数据
```
GET /api/statistics          # 统计数据
GET /api/events              # 事件列表
GET /api/news                # 新闻列表
GET /api/risk-indices        # 风险指数
GET /api/economic-data       # 经济数据
```

### 多维度数据 ⭐ NEW
```
GET /api/technology-data     # 科技数据
GET /api/environment-data    # 环境数据
GET /api/energy-data         # 能源数据
GET /api/food-data           # 粮食数据
GET /api/population-data     # 人口数据
GET /api/comprehensive-power-index  # 国力指数
```

## 📝 使用指南

### 本地运行
```bash
# 后端
cd backend
npm run dev

# 前端
cd frontend
npm run dev
```

### 访问地址
- 前端：http://localhost:3000
- 后端：http://localhost:3001
- WebSocket: ws://localhost:3001/ws

### 语言切换
1. 点击右上角语言选择器
2. 选择中文或 English
3. 界面立即切换

### 数据维度切换
1. 点击顶部标签页
2. 选择不同维度查看
3. 支持多标签对比

## 📊 数据更新

### 更新频率
- 实时数据：每 5 分钟
- 经济数据：每 10 分钟
- 多维度数据：静态（可定期更新）

### 数据来源
- 实时事件：生成 + GDELT
- 新闻：免费 API
- 经济数据：世界银行、UN
- 科技数据：WIPO、各国统计局
- 环境数据：世界银行、UNEP
- 能源数据：IEA、EIA
- 粮食数据：FAO
- 人口数据：UN Population Division
