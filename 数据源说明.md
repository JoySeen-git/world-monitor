### 3. 数据更新机制

**自动更新频率：**
- 每 10 分钟自动获取最新数据
- WebSocket 实时推送更新通知
- 数据自动去重

**数据来源：**
```typescript
// 新闻数据
- fetchChinaNews()      // 中国新闻
- fetchWorldNewsFromChina() // 国际新闻
- fetchMilitaryNews()   // 军事新闻
- fetchUSNews()         // 美国新闻
- fetchEUNews()         // 欧洲新闻

// 地缘政治事件
- generateDemoGeoEvents() // 实时地缘政治事件
```

## 🔧 技术实现

### 服务层架构
```
backend/src/services/
├── freeNewsService.ts    # 免费新闻 API 集成
├── rssService.ts         # RSS 源爬虫（备用）
└── geoEventService.ts    # 地缘政治数据（备用）
```

### 数据流程图
```
定时任务 (每 10 分钟)
    ↓
调用免费 API → 数据清洗 → 去重 → 存入数据库
    ↓
WebSocket 推送 → 前端实时更新
```

### 数据库结构
```json
{
  "events": [GeoEvent],    // 地缘政治事件
  "news": [NewsItem],      // 新闻
  "risk_indices": [RiskIndex] // 风险指数
}
```
