## 🚀 三种部署方式

### 方式一：使用一键部署脚本（最简单）

适合：Ubuntu/Debian 服务器

```bash
# 1. 上传项目到服务器
scp -r world-monitor user@your-server:/tmp/

# 2. 登录服务器
ssh user@your-server

# 3. 移动项目到部署目录
sudo mv /tmp/world-monitor /var/www/

# 4. 运行一键部署脚本
cd /var/www/world-monitor
sudo chmod +x deploy.sh
sudo ./deploy.sh
```

脚本会自动完成：
- ✅ 安装 Node.js 20
- ✅ 安装 PM2
- ✅ 安装 Nginx
- ✅ 配置防火墙
- ✅ 安装项目依赖
- ✅ 启动服务
- ✅ 配置 Nginx 反向代理
- ✅ 配置 HTTPS（如果提供域名）
- ✅ 设置自动备份

### 方式二：Docker 部署（最灵活）

适合：有 Docker 环境的用户

```bash
# 1. 创建 docker-compose.yml
cd world-monitor
nano docker-compose.yml

# 2. 启动服务
docker-compose up -d

# 3. 查看状态
docker-compose ps
docker-compose logs -f
```

### 方式三：手动部署（最可控）

适合：有经验的用户

```bash
# 1. 安装依赖
sudo apt update
sudo apt install nodejs npm nginx -y
sudo npm install -g pm2

# 2. 安装项目依赖
cd /var/www/world-monitor/backend
npm install
cd ../frontend
npm install

# 3. 启动服务
cd ../backend
pm2 start npm --name "wm-api" -- run dev
cd ../frontend
pm2 start npm --name "wm-frontend" -- run dev
pm2 save

# 4. 配置 Nginx
sudo nano /etc/nginx/sites-available/world-monitor
# （复制公网部署方案.md 中的配置）

sudo ln -s /etc/nginx/sites-available/world-monitor /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx

# 5. 配置 HTTPS
sudo apt install certbot python3-certbot-nginx -y
sudo certbot --nginx -d your-domain.com
```

## 🔧 常见问题

### 问题 1: 端口被占用
```bash
# 查看占用端口的进程
sudo lsof -i :3001
sudo lsof -i :3000

# 终止进程
sudo kill -9 <PID>
```

### 问题 2: Nginx 启动失败
```bash
# 检查配置
sudo nginx -t

# 查看错误日志
sudo tail -f /var/log/nginx/error.log
```

### 问题 3: PM2 服务异常
```bash
# 查看日志
pm2 logs

# 重启服务
pm2 restart all

# 重置进程
pm2 delete all
pm2 start all
```

### 问题 4: WebSocket 连接失败
检查 Nginx 配置中的 WebSocket 部分：
```nginx
location /ws {
    proxy_pass http://localhost:3001/ws;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection "Upgrade";
}
```

## 📈 性能优化建议

### 1. 启用 Gzip 压缩
```nginx
gzip on;
gzip_vary on;
gzip_min_length 1024;
gzip_types text/plain text/css text/xml text/javascript 
           application/x-javascript application/xml+rss 
           application/json application/javascript;
```

### 2. 设置缓存
```nginx
location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
    expires 1y;
    add_header Cache-Control "public, immutable";
}
```

### 3. 使用 CDN
- Cloudflare（免费）
- 阿里云 CDN
- 腾讯云 CDN
